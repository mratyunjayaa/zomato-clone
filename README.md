# Zomato Clone

A responsive restaurant discovery web app inspired by Zomato.

## Features

- Search Restaurants
- Filter by Rating
- Filter by Offers
- Open Now Filter
- Serves Alcohol Filter
- Price Sorting
- Distance Sorting
- Responsive Design

## Tech Stack

- HTML
- CSS
- JavaScript
